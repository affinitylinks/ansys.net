HYPER58

3-D 8-Node Mixed u-P Hyperelastic Solid

MP ME ST <> <> <> <> <> <> PP ED

HYPER58 Element Description

HYPER58 is used for 3-D modeling of solid hyperelastic structures. The mixed u-P (Displacement-Pressure) formulation allows the element matrices to be formed using mixed variational principles with the pressure introduced to enforce the incompressibility constraint. It is applicable to nearly incompressible rubber-like materials with arbitrarily large displacements and strains. The element is defined by eight nodes having three degrees of freedom at each node: translation in the nodal x, y, and z directions. The hyperelastic formulation is nonlinear and requires an iterative solution. Large deflections [NLGEOM] must be active in order to update the geometry each substep. See HYPER58 in the ANSYS, Inc. Theory Reference for more details about this element.

HYPER58 Input Data

The geometry, node locations, and the coordinate system for this element are shown in Figure 58.1: "HYPER58 Geometry". The element input data include eight nodes, the isotropic material properties, and the constants defining the Mooney-Rivlin strain energy function. The strain energy function may also be defined by user subroutine USRMOONEY and is accessed by KEYOPT(7) = 1. The Mooney-Rivlin function is valid for a wide range of nearly incompressible rubber and rubber-like materials. It is characterized by Poisson's ratio and constants aij (entered with TBDATA or *MOONEY commands), which may be a function of temperature. See Data Tables - Implicit Analysis for the TBDATA command input. The element coordinate system orientation is as described in Coordinate Systems.

Element loads are described in Node and Element Loads. Pressures may be input as surface loads on the element faces as shown by the circled numbers on Figure 58.1: "HYPER58 Geometry". Positive pressures act into the element.

Element body loads may be input as temperatures at the nodes. The node I temperature T(I) defaults to TUNIF. If all other temperatures are unspecified, they default to T(I). For any other input pattern, unspecified temperatures default to TUNIF.

A summary of the element input is given in "HYPER58 Input Summary". A general description of element input is given in Element Input.

HYPER58 Output Data

The solution output associated with the element is in two forms:

  • Nodal displacements included in the overall nodal solution

  • Additional element output as shown in Table 58.1: "HYPER58 Element Output Definitions".

Several items are illustrated in Figure 58.2: "HYPER58 Stress Output". KEYOPT(5) provides various element printout options (see Element Solution). The element stress directions are parallel to the element coordinate system. A general description of solution output is given in Solution Output. See the ANSYS Basic Analysis Guide for ways to view results.

The Element Output Definitions table uses the following notation:

A colon (:) in the Name column indicates the item can be accessed by the Component Name method [ETABLE, ESOL]. The O column indicates the availability of the items in the file Jobname.OUT. The R column indicates the availability of the items in the results file.

In either the O or R columns, Y indicates that the item is always available, a number refers to a table footnote that describes when the item is conditionally available, and a - indicates that the item is not available.

HYPER58 Assumptions and Restrictions

  • Zero volume elements are not allowed.

  • Elements may be numbered either as shown in Figure 58.1: "HYPER58 Geometry" or may have the planes IJKL and MNOP interchanged.

  • The element may not be twisted such that the element has two separate volumes. This occurs most frequently when the elements are not numbered properly.

  • All elements must have eight nodes. A prism-shaped element may be formed by defining duplicate K and L and duplicate O and P node numbers (see Triangle, Prism and Tetrahedral Elements). A tetrahedron shape is also available.

  • Material properties are evaluated at the average of the nodal temperatures.

HYPER58 Product Restrictions

There are no product-specific restrictions for this element.
