CONTAC26

2-D Point-to-Ground Contact

MP ME ST <> <> <> <> <> <> PP ED

CONTAC26 Element Description

CONTAC26 is a gap element representing a surface which will resist penetration by a node. The element is capable of supporting compression in the direction normal to the surface and shear (Coulomb friction) in the tangential direction. The element has two degrees of freedom at each node: translations in the nodal x and y directions. The surface may be straight or a circular arc. The surface may also be defined to move. Multiple elements may be defined to form a complex surface. The element may only be used to represent model-to-ground behavior; model-to-model behavior cannot be represented. Other contact elements, such as CONTAC12, COMBIN40, CONTAC48, CONTAC49, CONTAC52, etc. are also available.

The element may be initially preloaded in the normal direction or it may be given a gap specification. A specified stiffness acts in the normal direction when a node penetrates the surface. See CONTAC26 in the ANSYS, Inc. Theory Reference for more details about this element.

CONTAC26 Input Data

The geometry, node locations, and the coordinate system for this element are shown in Figure 26.1: "CONTAC26 Geometry". The element is defined by three nodes, surface constants, a normal stiffness (STIFN), and a stiffness for elastic Coulomb sliding (STIFS). Node I is the active node associated with the flexible model. Nodes J and K are used to define the ends of the contact surface, which may be defined as moving by imposed displacements. A straight surface (R = 0) is assumed to be a rectangle having its width between nodes J and K in the x-y plane and a semi-infinite length to the right of the surface line (when viewed from node J to node K).

A curved surface is assumed to be an arc defined by nodes J and K and a radius |R|. The arc is formed counterclockwise from J to K such that the angle swept by R is 180° or less. If KARC = 0, the surface extends over the counterclockwise arc from J to K. For KARC = 1, it extends over the counterclockwise arc from K to J. If R is positive, the region greater than R is assumed to be semi-infinite and to resist penetration. If R is negative, the pie-shaped region toward the center is assumed to resist penetration.

The stiffnesses should be based upon the stiffness of the surfaces in contact. See Contact of the ANSYS Structural Analysis Guide for guidelines on choosing a value for STIFN. In some cases (such as initial interference analyses, nonconvergence, or over penetration), it may be useful to change the STIFN value between load steps or in a restart in order to obtain an accurate, converged solution. The stiffness should be on a full 360° basis for an axisymmetric analysis. A default value of 1.0E9 is used for the STIFN, and the default for STIFS is STIFN/100.

The coefficient of friction µ is input as material property MU and is evaluated at the average of the two node temperatures.

A summary of the element input is given in Figure 26.1: "CONTAC26 Geometry". A general description of element input is given in Element Input.

CONTAC26 Output Data

The solution output associated with the element is in two forms:

  • nodal displacements included in the overall nodal solution

  • additional element output (for closed or changed status) as shown in Table 26.1: "CONTAC26 Element Output Definitions".

Several items are illustrated in Figure 26.2: "CONTAC26 Stress Output". For an axisymmetric analysis, the element forces are expressed on a full 360° basis. Solution Output gives a general description of solution output. See the ANSYS Basic Analysis Guide for ways to view results.

CONTAC26 Assumptions and Restrictions

  • The element is grounded so that only node I has any stiffness associated with it. Therefore, nodes J and K do not contribute to any reaction forces.

  • The 2-D contact surface element must be defined in an X-Y plane and the Y-axis must be the axis of symmetry for axisymmetric analyses. An axisymmetric structure should be modeled in the +X quadrants.

  • Nodes J and K should not be attached to other element types, otherwise, equilibrium will not be satisfied. The element operates bilinearly only in the static and the nonlinear transient dynamic analyses and is intended for use in a large-deflection analysis. If used in other analysis types, the element maintains its initial status throughout the analysis.

  • The element is nonlinear and requires an iterative solution. Nonconverged substeps are not in equilibrium.

  • A gap condition capability is also included in the reduced linear transient dynamic analysis (see GAP module in PREP7). Note that, although it is permissible to change STIFN, it is not permissible to change any other real constants between load steps. Therefore, if you plan to change STIFN, you cannot allow the value of STIFS to be defined by default, because the program would then attempt to redefine STIFS as STIFN changed. You must explicitly define STIFS whenever STIFN changes, to maintain a consistent value throughout all load steps.

  • The element may not be deactivated with the EKILL command.

  • The amount of the initial interference (or gap) is specified by the node and surface locations. The orientation of the gap changes for a curved surface option or if the contact surface rotates.

  • Unreasonably high stiffness values should be avoided. The rate of convergence decreases as the stiffness increases. When operating near the "corner" of a region, the loading should be small enough such that the node does not pass through the region (in the surface and out the side) in one substep.

  • Node I may be initially oriented anywhere inside or outside of the contact region (see Figure 26.1: "CONTAC26 Geometry"). Node I may enter or leave the contact region at any angle. The resisting forces are normal to the surface and, if friction is present, parallel to it. For a curved surface, the normal is along the line through the node and the center point. The initial gap (or interference) at the beginning of the first substep is determined from the node location normal to the surface. If the gap is open, no stiffness is associated with this element for this substep. If the gap is closed with either no friction (µ = 0.0 or KEYOPT(1) = 0) or sliding, the normal stiffness STIFN is used for gap resistance. If the gap is closed and stuck, the stiffness STIFS is also used for gap sticking resistance parallel to the surface.

  • In the normal direction, when the normal force FN is negative, the gap remains in contact and responds as a linear spring. As the normal force becomes positive, contact is broken and no force is transmitted. Elastic Coulomb friction is used if friction is defined, such that: |FS| < - µ*FN (if stuck, STAT = 1) or |FS| = - µ*FN (if sliding, STAT = +2). Frictionless contact is identified by STAT = +2.
